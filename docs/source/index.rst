Welcome to Focus Point's documentation!

Introduction
---------

Welcome to the documentation for Focus Point - your ultimate educational timetable app designed to help you stay organized and focused throughout your academic journey. Whether you're a student, educator, or anyone managing educational schedules, Focus Point has got you covered. This documentation provides a comprehensive guide on how to use Focus Point effectively.

Check out the :doc:`usage` section for further information, including
how to :ref:`install` the project.
